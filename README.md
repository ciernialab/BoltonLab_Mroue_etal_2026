# BoltonLab_Mroue_etal_2026

RNA-seq analysis of P8 microglia isolated from PVN of male and female pups after ELA or control.

2025-10-12_PVNSeq_finalcode.R: Analysis pipeline for identifying DEGs for ELA vs Control across and within each sex. Uses EdgeR and limma-voom

GeneEnrichmentTestingMouse.R: Fisher's exact test to compare DEG lists to known GR regulated gene lists

gsea.R: Gene set enrichment analysis for GO term and pathway enrichment testing
